# Python 身份证号码验证器

一个基于Python和PyQt5开发的身份证号码验证器应用，用于验证中国身份证号码的合法性和有效性。

## 功能特点

- 验证18位身份证号码的格式正确性
- 检查地区码的有效性
- 验证出生日期的合理性
- 计算并验证校验码
- 从有效的身份证号码中提取持证人信息：
  - 出生日期
  - 性别
  - 户籍所在地
- 友好的图形用户界面

## 系统要求

- Python 3.6+
- PyQt5

## 安装与运行

1. 克隆或下载本项目到本地
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 运行应用：
   ```
   python main.py
   ```

## 使用说明

1. 启动应用后，在输入框中输入18位身份证号码
2. 点击"验证"按钮进行验证
3. 查看验证结果和解析信息

## 技术实现

- **GUI框架**：PyQt5
- **验证逻辑**：
  - 身份证号码格式验证
  - 地区码对照表验证
  - 日期有效性验证
  - 校验码计算（GB 11643-1999标准）

## 应用场景

- 用户注册时的身份证信息验证
- 实名认证系统
- 身份证信息管理系统
- 数据清洗与校验
